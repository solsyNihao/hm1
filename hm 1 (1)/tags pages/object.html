< object >

Пример

Как понять?

Тег < object > используется для встраивания на страницу других страниц или мультимедийных элементов (PDF, Flash, видео- или аудиоплееров). Обычно поле type нужно указывать, но для YouTube-видео это необязательно. Если браузер не поддерживает тип встроенного файла, то он покажет вложенное содержимое в качестве фолбэка. В нашем примере, если встраивание PDF-инструкции не поддерживается, то мы увидим ссылку на её скачивание. Доступные атрибуты: data — ссылка на встраиваемый ресурс; type — MIME-тип подгружаемого объекта; name — имя объекта; usemap — атрибут для связи объекта с тегом < map >; width — ширина объекта; height — высота объекта; form — связь с одной или несколькими формами.
