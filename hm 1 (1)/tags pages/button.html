< button >

Пример

Как понять?

< button > создаёт кликабельную кнопку. С помощью CSS можно настроить внешний вид кнопки < button > и текста на ней: шрифт, размер, цвет фона и другие параметры. А если ничего не менять, то < button > будет выглядеть, как стандартная браузерная кнопка. На кнопку можно поместить любой HTML-элемент.
